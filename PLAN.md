## 各遊戲詳細規則

### 1. 五子棋 ✅
- 棋盤：15×15
- 規則：五子連線獲勝
- 當前：正常

### 2. 象棋 ⚠️ 需修改
```
棋盤：9×10格，中間「楚河漢界」
棋子：
- 將/帥：1顆，九宮格內走一步
- 士：2顆，斜走一格
- 象：2顆，田字走法（不過河）
- 車：2顆，直線任意
- 馬：2顆，日字走
- 炮：2顆，隔子打
- 兵/卒：5顆，前進不後退

勝利：將死對方
特殊：
- 將帥不能面對面
- 兵過河可左右走
```

### 3. 暗棋 ⚠️ 需修改
```
棋盤：4×8格
棋子：完整象棋32顆，全部翻面

玩法：
1. 輪流翻棋或吃棋
2. 翻出後才知道顏色與種類

吃棋規則（從大到小）：
將 > 士 > 象 > 車 > 馬 > 炮 > 兵

炮特殊：必須隔一子才能吃
兵特殊：可以吃將

勝利：吃掉對方所有棋子
```

### 4. 軍棋 ⚠️ 需修改
```
棋盤：6×6（2人對戰版）
棋子：30顆
- 司令 1
- 軍長 1
- 師長 1
- 旅長 1
- 團長 1
- 營長 1
- 連長 1
- 排長 1
- 工兵 1
- 地雷 3
- 炸彈 2
- 軍旗 1

吃棋規則：
- 大吃小
- 工兵挖地雷
- 炸彈同歸於盡
- 軍旗被吃即敗
```

### 5. 魔術方塊 ⚠️ 需修改
```
結構：3×3×3
- 6面
- 54格

操作：
- U（上）
- D（下）
- L（左）
- R（右）
- F（前）
- B（後）

目標：每面顏色一致

當前：簡化版，需完整還原
```

### 7. 踩地雷 ✅

### 8. 彈珠台 ✅

### 9. 接龍 ⚠️ 需優化
```
牌數：52張
目標：四個花色堆由A到K

規則：
- 桌面牌堆：黑紅交錯，大壓小
- 暫存堆：移動一張
- 花色堆：同花色由小到大
```

### 10. 大老二 ⚠️ 需優化

```
- 三條
- 順子
- 同花
- 葫芦
- 鐵支
- 同花順

大小：2最大，3最小
```
