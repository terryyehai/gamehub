# GameHub - 網頁遊戲平台

精選網頁遊戲平台，支援手機、平板、電腦遊玩。

## 🎮 遊戲清單

### 棋類
- ⭕ 五子棋 - 經典黑白對弈
- 🎲 象棋 - 傳統中國象棋
- 🌑 暗棋 - 翻轉吃子象棋
- ⚔️ 軍棋 - 四人軍棋大戰

### 益智
- 🧊 魔術方塊 - 3D 互動方塊
- 🐍 貪食蛇 - 經典貪吃蛇
- 💣 踩地雷 - 掃雷遊戲
- 🔵 彈珠台 - 物理彈珠

### 牌類
- 🃏 接龍 - 紙牌接龍
- 🃗 大老二 - 經典撲克牌

## 🚀 使用技術

- HTML5 / CSS3 / JavaScript (ES6+)
- Canvas API - 遊戲繪圖
- Web Audio API - 音效
- 響應式設計 - 支援手機/平板/電腦
- 本地存儲 - 遊戲進度儲存

## 📱 安裝使用

### 線上遊玩
訪問 GitHub Pages 即可遊玩：
```
https://terryyehai.github.io/gamehub/
```

### 本地運行
```bash
# 克隆專案
git clone https://github.com/terryyehai/gamehub.git

# 或使用 Python 簡單伺服器
cd gamehub
python -m http.server 8000

# 瀏覽器訪問
http://localhost:8000
```
